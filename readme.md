# MultiCoinCharts - Crypto/Stocks Graph Open Source
#### Josiah Leas & [PrettySights](https://prettysights.com)

### IMPORTANT 
Uses TradingView widgets to show many cryptocurrencies, stocks, and foreign exchange markets.
